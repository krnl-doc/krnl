**❌ Krnl says "Failed to inject"**
> Roblox likely just updated. Wait 24-48 hours for the Krnl team to push a patch, then re-download the latest bootstrapper.

**❌ Windows Defender deletes Krnl immediately**
> Add an exclusion folder before downloading. Go to Windows Security > Virus & threat protection > Manage settings > Add or remove exclusions. Create a folder like `C:\Krnl` and add it as an exclusion. Download Krnl into that folder.

**❌ Key doesn't work / says invalid**
> Keys expire after 24 hours. Generate a new one. If the problem persists, clear your browser cache and try a different browser for the Linkvertise steps.

**❌ Krnl crashes when executing large scripts**
> Increase your Windows page file size or close other applications. Large GUI scripts require more memory.

**❌ Script executes but nothing happens in-game**
> Make sure you injected before the game fully loaded. Rejoin the game, inject immediately after spawning, then execute.

**❌ "Unable to find Roblox" error**
> Launch Roblox manually first, then open Krnl. If that doesn't work, reinstall both Roblox and Krnl.

**❌ Is there a mobile version of Krnl?**
